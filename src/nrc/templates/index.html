Open Notifications is a modern, open-source component for routing messages between components, systems, and applications. It implements the standardized VNG Notifications API.

This component relies on an Authorizations API for arranging authorizations and can be used in combination with the APIs for Zaak work as implemented in Open Zaak.

Open Notifications is based on the source code of the reference implementation of VNG Realisatie to create a stable API that can be used in production by municipalities.
